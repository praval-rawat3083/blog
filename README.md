# Porsche Blog Website

A responsive blog website built using HTML and CSS.

## Features

* Responsive design for mobile, tablet, and desktop
* Blog-style layout
* Card-based article sections
* Clean and modern UI
* Optimized image layout

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
